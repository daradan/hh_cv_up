## Authorizing and upping CV on HH

### Installation and setup
Clone repositories
```
git clone https://github.com/daradan/hh_cv_up.git
```
Installing libraries
```
pip install -r requirements.txt
```
Create file ___.env___ and fill in your data
```
URL=... # url to your CV
EMAIL=...
PASSWORD=...
```
